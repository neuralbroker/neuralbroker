## Things I've built (2026)

**BlitzKode** — Fine-tuned Qwen2.5-Coder-1.5B on 2,000 instruction examples. Three training stages (SFT, reward SFT, DPO) plus GGUF export for CPU-only inference. 510+ downloads on Hugging Face. [Try it →](https://huggingface.co/neuralbroker/blitzkode) · [code →](https://github.com/neuralbroker/blitzkode)

**EsenceLab** — Full-stack AI hiring platform (student, recruiter, admin). Gemini-powered resume parsing cut manual shortlisting by 60% across 200+ resumes. Three-service monorepo on Vercel + Render. Won 2nd Prize at Innovision (30+ teams). [code →](https://github.com/neuralbroker/esencelab)

**Lexora AI** — Document Q&A with retrieval. Per-user FAISS indices, OpenAI for generation, Redis for hot cache, Celery for the slow stuff. Built because I got tired of searching 200-page research papers manually. [code →](https://github.com/neuralbroker/lexora-ai)

**SentinelML** — Credit-card fraud detection. RandomForest on 25 features, 0.94 ROC-AUC, SMOTE oversampling. Drift detection via PSI, weekly auto-retrain, MLflow tracking. The lightbulb moment was discovering that one hand-engineered feature (`txn_velocity_24h`) mattered more than anything the model learned. [code →](https://github.com/neuralbroker/sentinel-ml)
